# Room Sharing (mobile app & web app)

## project overview

A web and mobile application that connects users with house owners and people offering shared rooms. The platform allows users to search, browse, and contact house owners or room-sharing providers, while also enabling them to manage their profiles and create posts.

Administrators can manage users, monitor posts, and control platform activity.
The system is designed to help reduce the room and housing shortage by making accommodation discovery faster, organized, and accessible.

---

## Technologies Used

- **Frontend (Web):** React, Tailwind CSS / Mantine
- **Mobile App:** React Native(expo)
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Hosting:** Vercel / Netlify (frontend), Render / Railway (backend)
- **Authentication:** JWT or session-based auth

---

## User side

### Features:

#### Search & Filters

- Filter by category (rental / sharing)
- Filter by city, price range, and availability
- Quick search by keywords

#### Browse Listings

- View rental houses and shared rooms
- Grid or list layout options
- Overview details including city, price, and room capacity

#### User Account Management

- Register and login securely
- Edit profile information
- Upload profile photo

#### Post Management

- Create room rental or sharing posts
- Edit or delete existing posts
- Upload multiple images

#### Direct Chat / Messaging

- Real-time messaging between users and owners
- Conversation history saved

#### Notifications

- New messages
- Post inquiries
- System alerts

#### Share Listing

- Share room or rental posts via link
- Share through social media

#### Responsive Design

- Works smoothly on mobile, tablet, and desktop
- Optimized UI for both web and mobile apps

---

## Admin Side

### Features:

#### User Management

- View all registered users
- Block, activate, or delete accounts

#### Post Management

- Approve or remove listings
- Monitor inappropriate content
- View reports from users

#### Dashboard & Analytics

- Total users, posts, and active listings
- Platform activity overview

#### Moderation Tools

- Handle reported posts
- Manage spam or fake listings
